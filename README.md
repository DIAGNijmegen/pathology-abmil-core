# pathology-clam-addmil

## About
This repo consists of CLAM [1] (cloned on 20240822) and integrated with Additive MIL [2]. Code duplicated from Catherine's local copy at [3]

It is created to support a study:

Title: Automated detection of cutaneous squamous cell carcinoma (CSCC) in whole slide images of skin biopsies using a weakly-supervised learning approach

Authors: Chia C., MSc, Dooper S, MSc, Mooyaart A.L., MD PhD, Amir A., MD, PhD, Wakkee M., MD PhD, Litjens G., PhD
